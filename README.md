## Weather App

This repository contains a weather application that provides the following information:

• Temperature

• Sky Status

• Wind Speed

• Humidity

• City Name

Simply type the city name in the search box to get the latest weather information.

## Technologies Used

• HTML

• CSS

• JavaScript
